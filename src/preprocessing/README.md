# Preprocessing

Preprocessing module migrated from `dev/hs`.

## Run

```bash
python -m src.preprocessing
```

## Inputs/Outputs

- Input transcripts: `data/raw`
- Metadata CSV: `data/metadata/lecture_metadata.csv`
- Preprocessed transcripts: `data/preprocessed`
- Terminology dictionary: `src/preprocessing/resources/term_mapping_dict.json`
